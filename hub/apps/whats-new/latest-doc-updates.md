# <a name="latest-updates-to-the-windows-developer-docs"></a>Dernières mises à jour des documents des développeurs Windows

Voici quelques rubriques qui ont reçu récemment des mises à jour importantes en date du 2 juin 2020. 

Remarque : avec la publication de Windows 10 build 19041 et certaines restructurations systémiques de la documentation, ce mois-ci, la liste est *très* longue. Pour obtenir la liste spécifique des API ajoutées dans le cadre de la build 19041 de Windows 10, consultez [cette liste](https://docs.microsoft.com/windows/uwp/whats-new/windows-10-build-19041-api-diff).
